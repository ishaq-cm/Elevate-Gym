# **App Name**: Elevate Gym Elite

## Core Features:

- AI Path Strategist: An AI-powered tool that analyzes a member's fitness level and goals to recommend the ideal coach, program, and weekly schedule combination.
- Single-Session Interactive Router: A high-performance hash-based navigation system for fluid, multi-page transitions within a single immersive view.
- Kinetic Before/After Reveal: Custom mouse and touch-driven interaction components for comparing member transformations in high fidelity.
- Dynamic Slot Scheduling: A color-coded, real-time timetable featuring tooltips for session availability and rapid trainer lookup.
- Multi-Tenure Billing Matrix: An interactive pricing toggle allowing users to swap between Monthly, Quarterly, and Annual tiers with dynamic price updates.
- Curated Gallery Engine: A masonry-layout media center with custom filtering tabs and a swipe-ready lightbox for mobile and desktop.
- Lead Management System: Customized contact experience with inquiry categorization and immediate session feedback status.

## Style Guidelines:

- Primary color: High-Impact Red (#E02B2B) for energy and urgency. Background: Onyx Dark (#0A0707) which is a desaturated derivative of the red. Accent: Warm Ember (#B51F1F) to provide analogous depth.
- Headings: 'Bebas Neue' for an authoritative, athletic feel. Body: 'DM Sans' for readability and contemporary precision. Note: currently only Google Fonts are supported.
- Strict sharp-edged icons using Font Awesome 6, ensuring a brutalist and consistent architectural aesthetic.
- Full-bleed hero sections with zero-radius components, mirroring an elite, high-performance agency portfolio style.
- Custom cursor LERP following and staggered loader sequence, delivering a boutique digital experience on every refresh.
